# Triage Labels

Las skills hablan en términos de cinco roles canónicos de triage. Este archivo mapea esos roles a los strings de label realmente usados en este repo.

| Label en mattpocock/skills | Label en nuestro tracker | Significado                                  |
| -------------------------- | ------------------------ | -------------------------------------------- |
| `needs-triage`             | `needs-triage`           | Maintainer tiene que evaluar este issue      |
| `needs-info`               | `needs-info`             | Esperando más info del reporter              |
| `ready-for-agent`          | `ready-for-agent`        | Totalmente especificado, listo para AFK      |
| `ready-for-human`          | `ready-for-human`        | Requiere implementación humana               |
| `wontfix`                  | `wontfix`                | No se va a accionar                          |

Cuando una skill menciona un rol (ej. "aplicá el label AFK-ready"), usá el string correspondiente de la columna derecha.

Editá la columna derecha si más adelante adoptás otro vocabulario.
